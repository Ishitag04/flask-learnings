\## ▶️ How to Run



1\. Clone the repo:

&#x20;  git clone https://github.com/your-username/flask-learning.git



2\. Navigate to any folder:

&#x20;  cd 01\_login\_basic



3\. Install Flask:

&#x20;  pip install flask



4\. Run the app:

&#x20;  python -m flask run --debug



5\. Open browser:

&#x20;  http://127.0.0.1:5000/
